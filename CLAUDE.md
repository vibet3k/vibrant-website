# Claude Code Instructions

## Git Workflow
- Always commit and push directly to the main branch
- Do not create feature branches or pull requests
- Write clear, descriptive commit messages

## Project Context
- This is a Next.js website deployed on Vercel
- Vercel auto-deploys from the main branch
- Use existing code patterns and styling conventions
- Escape special characters in JSX (use &quot; for quotes, &apos; for apostrophes, etc.)

## Tech Stack
- Next.js 15 with App Router
- Tailwind CSS for styling
- Sanity CMS for blog content
- Vercel Analytics and HubSpot tracking enabled
